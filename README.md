# 使用

 1. 在本仓库下发起 Issues ，表明你的申请意愿同时注明你的仓库名，管理员批准后会为你新建一个仓库，并在对应的 Issue 给予你回复.
 2. Fork 你的仓库，上传图片到你的仓库中，完成后发起 Pull requests 申请，管理员审核后予以答复。

# 外链

你的图床根目录：

 - Github节点
 
  ```
  https://imageurl.github.io/<你的仓库名>/
  ```
  
 - jsDelivr节点

  ```
  https://cdn.jsdelivr.net/gh/imageurl/<你的仓库名>/
  ```
  
 # 不被允许的
 
 与上传图片无关的操作均不被允许。
